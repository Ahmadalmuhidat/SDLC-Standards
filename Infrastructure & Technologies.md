## Management
* **Microsoft 365** is a productivity cloud system that provides services to run the business. The provided services are Office 365, OneDrive, Excel, etc. ( ***paid*** )
* **Jira** is used for managing projects through tracking progress, and issues, and assigning tasks to the team members.
Jira can be used in every phase of the software life cycle phases.( ***paid*** )
* **cPanel** is an online Linux-based graphical interface (GUI) used as a control panel to simplify website and server management. cPanel allows to publish websites, manage domains, organize web files, create email accounts, etc.( ***paid*** )
* **Microsoft Projects**  is project management software that's used to create schedules, and project plans, manage resources and keep track of time.( ***paid*** )
* **Microsoft Visio** is software for drawing a variety of diagrams. These include flowcharts, org charts, building plans, floor plans, data flow diagrams, and many more.( ***paid*** )
* **Pipedrive**  is a Web-based Sales CRM and pipeline management solution that enables businesses to plan their sales activities and monitor deals.( ***paid*** )

## Hosting
* **Cloudflare** is a global network designed to make everything we connect to the Internet secure, private, fast, and reliable. Secure our websites, APIs, and Internet applications.
We use Cloudflare for hosting web applications and maintaining traffic and security.( ***paid*** )
* **Heroku** is a cloud service that is used to deploy, manage, and scale modern apps. Heroku supports several programming languages & frameworks, also we can use it to deploy SQL/NoSQL databases.
We use Heroku for hosting API, databases, and web applications.( ***paid*** )
* **Github** is a code hosting platform for version control and collaboration. It allows us to work together on projects from anywhere.( ***free*** )
